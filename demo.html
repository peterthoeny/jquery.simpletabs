Demo of jquery.simpletabs jQuery Plugin

The jquery.simpletabs plugin shows a tabs bar for navigation. The tabs can be defined once, and shared across multiple HTML pages.

This demo has two pages, demo.html (this page), and demo2.html - click on a tab to switch.

The tabs can be styled, see default jquery.simpletabs.css.
